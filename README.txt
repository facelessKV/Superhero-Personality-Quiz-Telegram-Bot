# Инструкция по установке и запуску Telegram-бота "Кто ты из супергероев?"
### Подробная инструкция для начинающих

## Установка на Windows

### Шаг 1: Установка Python 3.9
1. Скачайте Python 3.9.7 по ссылке: https://www.python.org/downloads/release/python-397/
   - Прокрутите страницу вниз и выберите "Windows installer (64-bit)"
   - Если у вас старый компьютер, возможно, вам нужна 32-битная версия

2. Запустите скачанный файл
   - ОБЯЗАТЕЛЬНО поставьте галочку "Add Python 3.9 to PATH"
   - Нажмите "Install Now"

3. Дождитесь окончания установки и нажмите "Close"

### Шаг 2: Скачивание и подготовка бота
1. Создайте на рабочем столе новую папку с названием "superhero-bot"
2. Скопируйте файл с кодом бота в эту папку
3. Переименуйте файл в "bot.py", если он называется иначе

### Шаг 3: Создание файла с токеном
1. Откройте Блокнот (нажмите Win+R, введите notepad и нажмите Enter)
2. Напишите в нём:
   ```
   BOT_TOKEN=ваш_токен_от_botfather
   ```
   Вместо "ваш_токен_от_botfather" вставьте токен, который вы получили от @BotFather в Telegram
3. Сохраните этот файл в папку "superhero-bot" под именем ".env"
   - В диалоговом окне "Сохранить как" выберите тип файла "Все файлы (*.*)"
   - В поле имени введите ".env" (с точкой в начале)
   - Убедитесь, что вы сохраняете в папку "superhero-bot"

### Шаг 4: Установка необходимых библиотек и запуск бота
1. Нажмите Win+R, введите "cmd" и нажмите Enter
2. В открывшейся командной строке введите следующие команды по очереди:
   ```
   cd Desktop\superhero-bot
   python -m pip install --upgrade pip
   pip install aiogram==3.3.0 python-dotenv
   python bot.py
   ```

3. Бот запустится, и вы увидите сообщение о его активации
4. Чтобы остановить бота, нажмите Ctrl+C в командной строке

## Установка на Linux

### Шаг 1: Установка Python 3.9
Откройте терминал (Ctrl+Alt+T в большинстве дистрибутивов) и введите:

```
sudo apt update
sudo apt install python3.9 python3.9-venv python3-pip
```

### Шаг 2: Создание папки для бота
В терминале введите:
```
mkdir ~/superhero-bot
cd ~/superhero-bot
```

### Шаг 3: Создание файла бота
1. Введите в терминале:
   ```
   nano bot.py
   ```

2. Скопируйте весь код бота и вставьте его в открывшееся окно редактора
   - Для вставки в терминале обычно используется Ctrl+Shift+V или правый клик мыши

3. Сохраните файл, нажав Ctrl+O, затем Enter, и выйдите из редактора с помощью Ctrl+X

### Шаг 4: Создание файла с токеном
1. Введите в терминале:
   ```
   nano .env
   ```

2. Напишите:
   ```
   BOT_TOKEN=ваш_токен_от_botfather
   ```
   Вместо "ваш_токен_от_botfather" вставьте токен, который вы получили от @BotFather в Telegram

3. Сохраните файл (Ctrl+O, затем Enter) и выйдите из редактора (Ctrl+X)

### Шаг 5: Установка необходимых библиотек и запуск бота
В терминале введите:
```
pip3 install aiogram==3.3.0 python-dotenv
python3 bot.py
```

Бот запустится, и вы увидите сообщение о его активации.
Чтобы остановить бота, нажмите Ctrl+C в терминале.

## Как получить токен для бота

1. Откройте Telegram и найдите @BotFather
2. Напишите ему команду /newbot
3. Следуйте его инструкциям:
   - Введите имя бота (как он будет отображаться в контактах)
   - Введите имя пользователя для бота (должно заканчиваться на "bot")
4. BotFather пришлет вам токен - длинную строку символов
5. Скопируйте этот токен и вставьте его в файл .env как описано выше

## Использование бота

1. После запуска найдите своего бота в Telegram по имени пользователя, которое вы дали ему
2. Отправьте боту команду /start, чтобы начать тест
3. Отвечайте на вопросы, нажимая на кнопки с вариантами ответов
4. После ответа на все вопросы вы получите результат
5. Чтобы пройти тест заново, используйте команду /restart

## Решение проблем

Если бот не запускается или возникают ошибки:

1. Убедитесь, что вы правильно ввели токен в файл .env
2. Проверьте, что файл .env находится в той же папке, что и bot.py
3. Убедитесь, что вы установили все необходимые библиотеки
4. Если вы видите ошибку "ModuleNotFoundError", выполните установку библиотек еще раз

При возникновении других проблем попробуйте перезапустить компьютер и повторить процесс установки.
